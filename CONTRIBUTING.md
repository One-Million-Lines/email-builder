# Contributing

Thanks for your interest in contributing.

## Local setup

```bash
npm install
npm run dev
```

## Making changes

- Keep public API changes reflected in `src/index.ts` and the README
- Run `npm run build` and `npm run lint` before opening a pull request
- Include sample JSON or screenshots when you change editor behavior

## Pull requests

1. Fork the repository
2. Create a branch for your change
3. Make your changes
4. Open a pull request with a short description of the update

## Reporting issues

Open a GitHub issue with reproduction steps, expected behavior, and sample document JSON when relevant.
